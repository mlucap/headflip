# Headflip
Easy coin toss. Allows for results in heads or tails, true and false, or 1 and 0.

## Installation
```
npm i headflip -g --save
```

## Usage
```
$ flip
> tails
```
or

```js
const headflip = require('headflip')

headflip()
// heads or tails

headflip(num)
// 1 or 0

headflip(bool)
// true or false

```
