# Headflip
Easy coin toss. Allows for results in heads or tails, true and false, or 1 and 0.

## Installation
```
npm i headflip --save
```

## Usage
```js
const headflip = require('headflip')

console.log(headflip())
// heads or tails

console.log(headflip('num'))
// 1 or 0

console.log(headflip('bool'))
// true or false

```
